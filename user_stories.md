Admin User Stories
Manage user accounts  
As an Admin, I want to create, update, and delete user accounts so that I can control access to the system.

Assign roles  
As an Admin, I want to assign roles (Doctor, Patient, Admin) so that users have the correct permissions.

Monitor system activity  
As an Admin, I want to view logs and reports so that I can track usage and detect issues.

Configure system settings  
As an Admin, I want to update system configurations so that the application remains secure and efficient.

Patient User Stories
Book appointments  
As a Patient, I want to book appointments with doctors so that I can receive timely medical care.

View appointment history  
As a Patient, I want to see my past and upcoming appointments so that I can manage my schedule.

Access prescriptions  
As a Patient, I want to view my prescriptions online so that I can follow my treatment plan.

Update personal information  
As a Patient, I want to edit my profile details so that my records remain accurate.

Doctor User Stories
Manage availability  
As a Doctor, I want to set my available time slots so that patients can book appointments accordingly.

View patient appointments  
As a Doctor, I want to see my scheduled appointments so that I can prepare for consultations.

Update prescriptions  
As a Doctor, I want to add or update patient prescriptions so that treatment records are accurate.

Access patient records  
As a Doctor, I want to view patient medical histories so that I can provide informed care.
